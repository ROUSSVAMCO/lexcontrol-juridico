USUARIO ADMINISTRADOR INICIAL
Usuario: admin
Contraseña: Admin123*

Este usuario NO se registra. El sistema lo crea automáticamente la primera vez que se ejecuta.

QUÉ PUEDE HACER EL ADMINISTRADOR:
- Crear usuarios hasta un máximo de 15.
- Cambiar nombre del sistema.
- Cambiar subtítulo.
- Cambiar imagen de portada.
- Cambiar logo/foto de perfil del sistema.
- Cambiar colores.
- Ver todos los expedientes.
- Crear y compartir expedientes.

USO EN WINDOWS:
1. Instala Python desde https://www.python.org/downloads/
   Marca la casilla Add Python to PATH.

2. Abre esta carpeta en Visual Studio Code.

3. En la terminal ejecuta:
   pip install -r requirements.txt

4. Ejecuta:
   python app.py

5. Abre Chrome:
   http://127.0.0.1:5000

IMPORTANTE:
Después del primer acceso, entra con admin y cambia la contraseña editando el código o creando otro usuario administrador.
